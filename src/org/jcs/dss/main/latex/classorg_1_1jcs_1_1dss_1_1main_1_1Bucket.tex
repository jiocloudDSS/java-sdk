\subsection{Detailed Description}
Represents a D\+SS bucket. 

Every object stored in J\+CS D\+SS is contained within a bucket. . Within a bucket, any name can be used for objects. However, bucket names must be unique across all of J\+CS D\+SS. Ownership of the bucket is retained as long as the owner has an J\+CS D\+SS account.
